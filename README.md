#Project expenses in Python

## What does it do?
The program has a console-based menu with four options: enter an expense, view a specific expense, delete an expense, and exit the program

## Folder Structure
  main.py is where all the data is entered and displayed via the console
  gasto.py is where the "Gasto" class is declared
  proccess_gasto.py is where all the magic begins, here are all the functions for operate.

## Setup Steps

### 1.- Clone Repository

```
git clone <repo-url>
```
### 2.- Open the project
Open the cloned repository in your preferred code editor.
### 3.- Run the project
From the project root, execute the follow command
```
python3 main.py
```
